# If / Else if / Else

```javascript
let a = true;

if (condition) {
    // instructions
} else if {
    // instructions
} else {
    // instructions
}
```

**Comparative expressions**
- ```if (a)``` means ```if (a === true)```
- ```if (!(a))``` means ```if (a === false)```
- ```if (numberOfGuests === numberOfSeats)```

* true  + true = true
* true  + false = false BUT true or false = true
* false + false = false
* !true = false
* !false = true
