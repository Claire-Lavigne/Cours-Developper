## Types

### Numbers
* Integers : 1 ou -1
* Float : 1.5 ou -1.5
```javascript
let price = 2599;   // for e-commerce : price = 2599 not 25.99
```
:warning: Float à éviter

### Booleans
```javascript
let userSignedIn = true;
let checkboxIsChecked = false;
```
  
### String
```javascript
let firstName = 'Pedro';
let lastName = "Sanchez";

// concatenate
console.log(firstName + ' ' + lastName);
```
